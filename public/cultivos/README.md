# Imágenes de cultivos

Poné acá las fotos de cada cultivo. Aparecen automáticamente en la ficha del lote
y en las tarjetas flotantes del mapa cuando el lote tiene ese cultivo asignado.

## Cómo usarlo

- El nombre del archivo es el del cultivo **en minúsculas, sin tildes**.
- Formatos aceptados (se prueban en este orden): **.jpg, .jpeg, .png, .webp**.
- Si no hay imagen para un cultivo, se usa un degradado con el color del cultivo (no rompe nada).

## Nombres exactos esperados

| Cultivo  | Archivo (cualquiera de estas extensiones) |
|----------|--------------------------------------------|
| Soja     | `soja.jpg`                                 |
| Maíz     | `maiz.jpg`                                 |
| Trigo    | `trigo.jpg`                                |
| Girasol  | `girasol.jpg`                              |
| Cebada   | `cebada.jpg`                               |
| Alfalfa  | `alfalfa.jpg`                              |
| Sorgo    | `sorgo.jpg`                                |
| Avena    | `avena.jpg`                                |
| Trébol   | `trebol.jpg`                               |

Para cualquier otro cultivo, el archivo es el nombre en minúsculas sin tildes y con
guiones en lugar de espacios (ej.: "Maíz Dulce" → `maiz-dulce.jpg`).

Tamaño recomendado: ~800×800 px (cuadrada), formato JPG optimizado.
